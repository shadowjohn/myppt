# myppt
評選簡報致勝方程式

目前需要的環境如下：
1. Linux Server (Ubuntu or Fedora 都可以)
2. Requires:
      mariadb or mysql 任何版本
      libreoffice >= 4.2.8.2
      poppler-utils >= 0.24 (為了用 pdftotext 功能)
      xorg-x11-server-Xvfb >= 1.14.4
      php >= 5.3
      ImageMagick >= 6.8.6.3 (為了 PDF -> PNG)
      apache >= 2.4
3. 預設安裝路徑：/var/www/ppt
　　請詳見 inc/config.php、inc/conn.php
